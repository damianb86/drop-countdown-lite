# Product brief

## Problema
Merchants want Launchpad-like upcoming release experiences without full Plus campaign tooling.

## Usuario objetivo
Brand owner running limited drops, pre-release announcements or seasonal collection launches.

## Merchant persona
Merchant con catalogo activo, equipo chico y necesidad de resolver un problema operativo puntual sin instalar una suite pesada.

## Flujo principal
Select upcoming products/collections, confirm publish timing, preview countdown cards and publish storefront block later.

## Propuesta de valor
A small drop planning app that turns scheduled publish dates into clean upcoming-drop screens.

## MVP
- Dashboard of upcoming, live and expired drops.
- Drop planner with timing, product list and countdown preview.
- Settings for time zone, sold-out message and publish readiness.
- Empty state for stores without scheduled releases.

## Futuro fuera de alcance
- Theme block for upcoming product grid.
- Notify-me capture with consent logging.
- Automatic state switch on publish time.
